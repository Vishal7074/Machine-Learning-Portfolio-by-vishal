# 🎯 Feature Selection in Machine Learning

This repository demonstrates some of the most important **Feature Selection techniques** used in Machine Learning. Instead of using every available feature, Feature Selection helps identify the most informative ones, resulting in simpler, faster, and more accurate models.

The notebook focuses on both the **theoretical concepts** and **practical implementation** using **Python** and **scikit-learn**, making it suitable for beginners as well as intermediate Machine Learning learners.

---

## 📚 Topics Covered

### ✅ 1. Removing Duplicate Features

* Understanding duplicate columns
* Why duplicate features create unnecessary complexity
* Identifying and removing duplicate features

---

### ✅ 2. Variance Threshold

* What low variance means
* Why low-variance features are often uninformative
* Applying `VarianceThreshold`
* Choosing an appropriate threshold
* Advantages and limitations

---

### ✅ 3. Correlation-Based Feature Selection

* Correlation Matrix
* Identifying highly correlated features
* Removing redundant features
* Correlation heatmap interpretation
* Advantages and disadvantages of correlation-based selection

---

### ✅ 4. ANOVA (Analysis of Variance)

* Introduction to ANOVA F-Test
* Feature importance using statistical testing
* Understanding F-Score
* Selecting the best features using `SelectKBest`
* Practical implementation with `f_classif`

---

### ✅ 5. Logistic Regression Evaluation

* Training a Logistic Regression model
* Comparing model performance after feature selection
* Understanding how feature selection impacts prediction accuracy
