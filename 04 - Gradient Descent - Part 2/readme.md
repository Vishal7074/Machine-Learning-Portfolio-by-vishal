# Linear Regression: Sklearn vs Gradient Descent (From Scratch)

using two different approaches:

1. Scikit-learn's built-in `LinearRegression`
2. A custom **Gradient Descent Regressor implemented from scratch**

---

- **Diabetes Dataset** from `sklearn.datasets`
- 10 numerical input features
- Continuous target variable

---

### ✔ Linear Regression using Scikit-learn
